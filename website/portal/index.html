一句话介绍

独立完成从 Rhino 拓扑导出、数据 QC、异构图 + CVAE 建模到条件生成评估的完整流水线， 探索「用地 + 功能清单 → 采光/模数约束下的 3D 住宅体块」这一计算设计问题。

我的贡献

  • Rhino 数据管线 — 设计并实现 V14 导出脚本，提取房间邻接、采光面与拓扑 QC 规则
  • 离线校验 — 编写 JSON 数据集校验脚本，验证连通性、模数与神经网络张量转换
  • 模型训练 — 实现 SpatialModalCVAE（异构图编码 + 条件潜空间 + 体素解码）
  • 评估体系 — 建立 mIoU、功能匹配、模数合规率等 MVP 量化指标
  • 工程化 — 搭建可复现仓库结构与项目主页（本网站）
