## 2. 品牌名称冲突（重要）

### GraphSpace 英文名

已存在多个同名或近似品牌，领域不同但英文检索会混淆：

| 现有品牌 | 网址 | 领域 | 风险 |
|----------|------|------|------|
| GraphSpace（Virginia Tech） | [graphspace.org](https://graphspace.org) | 生物信息学网络可视化 | 英文 SEO / 学术检索冲突 |
| GraphSpace Inc. | [graphspace.dev](https://graphspace.dev) | ML 可视化开发环境 | 科技品牌冲突 |
| HugeGraph graph space | Apache HugeGraph 文档 | 图数据库多租户概念 | 技术术语层面 |

**影响评估：**

- 国内 B2B 主推「模态户型」可规避大部分冲突
- 论文与 GitHub 继续用 GraphSpace 可行，但英文摘要建议加注副标题（residential layout generation）
- 若 `graphspace.com` 不可用，**不影响**以 `.cn` 为主站的策略

### 中文「图空间」

- 作为描述性词汇（图 + 空间）较通用，商标显著性可能偏弱
- 建议在 42 类注册时配合 Logo 或组合标识（如「图空间 GraphSpace」）

### 中文「模态户型」

- 公开检索未发现完全一致的已注册商标
- 与「户型图」「智能户型」等品类词相近，注册前需在 [国家知识产权局商标检索](https://sbj.cnipa.gov.cn/sbcx/) 做近似查询
- 建议注册类别：**第 42 类**（软件/design）、**第 35 类**（推广）、**第 9 类**（可下载软件，如有客户端）

## 3. 注册前实时查询步骤

1. 打开 [阿里云域名查询](https://wanwang.aliyun.com/domain/) 或 [腾讯云域名注册](https://dnspod.cloud.tencent.com/)
2. 依次查询：`graphspace.cn`、`motaihuxing.cn`、`modalplan.cn`
3. 显示「可注册」则加入购物车；显示「已被注册」则查看 whois 是否可购买
4. 商标：登录 CNIPA 检索「图空间」「模态户型」，重点看 42/35/9 类
5. 微信公众号名称与产品名一致性预查（微信公众平台）
